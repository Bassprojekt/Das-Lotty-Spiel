## Key Game Mechanics (matching original)

1. **Peek technique**: Partially scratch zone corners to spot trap symbols before committing
2. **Trap triggers**: Only penalize when FULLY revealed - discard before that
3. **Catalogue progression**: Each prestige unlocks the next catalogue
4. **Final Chance**: Prestige trigger card at end of each catalogue
5. **Auto Scratcher**: Queue-based idle automation (can't peek, so only safe cards)
6. **Luck stat**: Reduces trap frequency, increases jackpot probability
